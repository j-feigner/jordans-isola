BGG Rating Tool

Compare and correlate user ratings in the BoardGameGeek database

  • Interfaces with the BGG XML API for pulling user collection data.
  • Analyzes data with both the Pearson and Spearman Ranked correlation coefficients to determine similarity in users' tastes in games.
  • Provides lists of games two users mutually like/dislike or agree/disagree about based off of their normalized scores.
